# Water Quality Data Analysis

## Overview
This project analyzes water quality data to identify trends and potential risks affecting public health.

## Objectives
- Assess water quality indicators
- Identify contamination patterns
- Visualize water quality trends
- Generate recommendations for improving water safety

## Tools Used
- Excel
- SQL
- Python
- GitHub

## Key Metrics
- pH
- Turbidity
- Dissolved Oxygen
- Total Coliforms

## Findings
Analysis showed variations in water quality across different sampling locations, highlighting areas requiring further monitoring.
